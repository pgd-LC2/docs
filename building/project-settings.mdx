## Accessing project settings

Click the settings icon in the editor to access all project configuration options.

## General settings

### Project information

Configure basic project details:

- **Project name** - Identify your project
- **Description** - Add notes about the project
- **Tags** - Organize projects with labels
- **Visibility** - Set to public or private

### Notifications

Control notification preferences:

- **Build completion** - Alert when builds finish
- **Deployment status** - Notify on publish success/failure
- **Error alerts** - Get notified of critical errors
- **Sound effects** - Enable/disable audio notifications

## Domain settings

### Primary domain

Set your website's main URL:

1. Choose a subdomain (e.g., `mysite.aisitebuilder.app`)
2. Or connect a custom domain
3. Set as primary domain

### Domain aliases

Add additional domains that redirect to your primary domain:

- Add multiple custom domains
- Configure redirects
- Manage SSL certificates

<Tip>
  SSL certificates are automatically generated and renewed for all domains.
</Tip>

## Hosting settings

### Deployment

Configure how your website is deployed:

- **Auto-deploy** - Automatically publish changes
- **Manual deploy** - Require manual approval
- **Branch deployment** - Deploy from specific branches

### Performance

Optimize your website performance:

- **CDN** - Enable content delivery network
- **Caching** - Configure cache settings
- **Compression** - Enable gzip/brotli compression
- **Image optimization** - Automatic image compression

## Analytics

### Built-in analytics

Track your website visitors:

- **Page views** - See which pages are popular
- **Visitor sources** - Understand where traffic comes from
- **Geographic data** - See visitor locations
- **Device types** - Desktop vs mobile traffic

### External analytics

Integrate third-party analytics:

- Google Analytics
- Plausible
- Fathom
- Custom tracking scripts

## Authentication

### User authentication

Enable user accounts on your website:

- **Email/password** - Traditional authentication
- **Social login** - Google, GitHub, etc.
- **Magic links** - Passwordless email login
- **Two-factor authentication** - Enhanced security

### Access control

Restrict access to your website:

- **Password protection** - Require password to view
- **IP whitelist** - Allow specific IP addresses
- **Member-only** - Require user account

## Database

### Database configuration

Manage your website's database:

- **View tables** - See all database tables
- **Add tables** - Create new tables
- **Edit schema** - Modify table structure
- **Export data** - Download database contents

### Backups

Protect your data:

- **Automatic backups** - Daily backups enabled by default
- **Manual backups** - Create backup snapshots
- **Restore** - Revert to previous backup

## File storage

### Storage settings

Configure file uploads and storage:

- **Storage limit** - View available storage
- **Allowed file types** - Restrict upload types
- **Max file size** - Set upload size limits
- **CDN delivery** - Serve files via CDN

### File management

Organize uploaded files:

- Browse all uploaded files
- Create folders
- Delete unused files
- View storage usage

## Environment variables

### Secrets management

Store sensitive configuration securely:

- **API keys** - Store third-party API keys
- **Database credentials** - Connection strings
- **OAuth secrets** - Authentication credentials
- **Custom variables** - Any sensitive data

<Warning>
  Never commit secrets to your code. Always use environment variables.
</Warning>

## Version history

### Rollback and restore

Manage your website versions:

- **View history** - See all previous versions
- **Compare versions** - See what changed
- **Restore version** - Revert to previous state
- **Preview version** - View without restoring

### Backup strategy

Create strategic backups:

- Before major changes
- After completing features
- Before experiments
- Regular intervals

## Advanced settings

### Custom code

Add custom functionality:

- **Custom CSS** - Override styles
- **Custom JavaScript** - Add functionality
- **Head tags** - Meta tags, scripts
- **Body tags** - Analytics, widgets

### SEO configuration

Optimize for search engines:

- **Meta title** - Page title
- **Meta description** - Page description
- **Open Graph** - Social media previews
- **Sitemap** - Auto-generated sitemap
- **Robots.txt** - Search engine instructions

### Performance monitoring

Track website performance:

- **Core Web Vitals** - Google performance metrics
- **Load times** - Page speed tracking
- **Error tracking** - Monitor errors
- **Uptime monitoring** - Track availability

## Team settings

### Collaboration

Work with team members:

- **Invite members** - Add team members
- **Roles and permissions** - Control access
- **Activity log** - See who changed what
- **Comments** - Discuss changes

<Note>
  Team features are available on Pro and Enterprise plans.
</Note>
