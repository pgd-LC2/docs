## Available AI agents

AISiteBuilder supports multiple AI agents, each with unique strengths. Choose the agent that best fits your project needs.

### Claude

Claude excels at understanding complex requirements and creating well-structured websites.

**Best for:**
- Complex multi-page websites
- Detailed customization
- Sophisticated layouts
- Content-heavy sites

**Strengths:**
- Excellent at following detailed instructions
- Strong understanding of design principles
- Great for iterative refinement

### GPT-4

GPT-4 provides fast, reliable website generation with broad capabilities.

**Best for:**
- Quick prototypes
- Standard website patterns
- General-purpose building
- Rapid iteration

**Strengths:**
- Fast response times
- Versatile across different website types
- Good balance of speed and quality

### Gemini

Gemini offers creative solutions and modern design approaches.

**Best for:**
- Creative and unique designs
- Modern web applications
- Experimental features
- Visual-heavy websites

**Strengths:**
- Creative design suggestions
- Modern web technologies
- Innovative solutions

## Selecting an agent

### In the editor

1. Click the agent selector dropdown
2. Choose your preferred agent
3. Start building or continue your project

<Note>
  You can switch agents mid-project. Each agent will work with the existing code.
</Note>

### Agent recommendations

The platform may recommend an agent based on your project:

- **Simple landing page** → GPT-4 for speed
- **Complex web app** → Claude for structure
- **Creative portfolio** → Gemini for design

## Working with agents

### Effective communication

All agents respond best to clear, specific instructions:

<CodeGroup>

```text Good prompt
Create a landing page for a SaaS product with:
- Hero section with headline, subheadline, and CTA button
- 3-column feature grid with icons
- Pricing table with 3 tiers
- FAQ accordion with 5 questions
- Footer with social links
Use a modern, professional design with blue as the primary color.
```

```text Vague prompt
Make a website for my business.
```

</CodeGroup>

### Plan mode with agents

Use plan mode to help agents understand your complete vision:

1. Select your agent
2. Enable plan mode
3. Describe your entire website
4. Review the agent's plan
5. Provide feedback
6. Approve and build

### Iterating with agents

Make changes through conversation:

- "Add a testimonials section after the features"
- "Change the color scheme to green and white"
- "Make the hero section taller"
- "Add animation to the CTA button"

## Tips for working with agents

<AccordionGroup>
  <Accordion title="Try different agents">
    If you're not getting the results you want, try switching to a different agent. Each has unique approaches.
  </Accordion>

  <Accordion title="Be patient with complex requests">
    Complex websites take time. Break large projects into smaller steps for better results.
  </Accordion>

  <Accordion title="Provide examples">
    Reference existing websites or designs to help agents understand your vision: "Create a hero section similar to stripe.com"
  </Accordion>

  <Accordion title="Use plan mode for big projects">
    For websites with many pages or features, use plan mode to ensure the agent understands the full scope.
  </Accordion>
</AccordionGroup>
