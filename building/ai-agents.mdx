## 可用的 AI 代理

AISiteBuilder 支持多个 AI 代理，每个都有独特的优势。选择最适合您项目需求的代理。

### Claude

Claude 擅长理解复杂需求并创建结构良好的网站。

**最适合：**
- 复杂的多页网站
- 详细的自定义
- 复杂的布局
- 内容丰富的网站

**优势：**
- 出色地遵循详细说明
- 对设计原则有很强的理解
- 非常适合迭代改进

### GPT-4

GPT-4 提供快速、可靠的网站生成，具有广泛的功能。

**最适合：**
- 快速原型
- 标准网站模式
- 通用构建
- 快速迭代

**优势：**
- 快速响应时间
- 适用于不同类型的网站
- 速度和质量的良好平衡

### Gemini

Gemini 提供创意解决方案和现代设计方法。

**最适合：**
- 创意和独特的设计
- 现代 Web 应用程序
- 实验性功能
- 视觉丰富的网站

**优势：**
- 创意设计建议
- 现代 Web 技术
- 创新解决方案

## 选择代理

### 在编辑器中

1. 点击代理选择器下拉菜单
2. 选择您首选的代理
3. 开始构建或继续您的项目

<Note>
  您可以在项目中途切换代理。每个代理都将使用现有代码。
</Note>

### 代理推荐

平台可能会根据您的项目推荐代理：

- **简单落地页** → GPT-4 速度快
- **复杂 Web 应用** → Claude 结构好
- **创意作品集** → Gemini 设计好

## 与代理合作

### 有效沟通

所有代理对清晰、具体的说明反应最好：

<CodeGroup>

```text 好的提示
为 SaaS 产品创建落地页，包含：
- 带有标题、副标题和 CTA 按钮的英雄区块
- 带有图标的 3 列功能网格
- 包含 3 个层级的价格表
- 包含 5 个问题的 FAQ 手风琴
- 带有社交链接的页脚
使用现代、专业的设计，蓝色为主色调。
```

```text 模糊的提示
为我的业务制作一个网站。
```

</CodeGroup>

### 与代理一起使用计划模式

使用计划模式帮助代理理解您的完整愿景：

1. 选择您的代理
2. 启用计划模式
3. 描述您的整个网站
4. 审查代理的计划
5. 提供反馈
6. 批准并构建

### 与代理迭代

通过对话进行更改：

- "在功能后添加推荐区块"
- "将配色方案更改为绿色和白色"
- "使英雄区块更高"
- "为 CTA 按钮添加动画"

## 代理能力

- 生成 HTML、CSS 和 JavaScript
- 创建响应式设计
- 实现现代 Web 功能
- 与数据库集成
- 设置身份验证
- 部署网站

## 与代理合作的技巧

  <Accordion title="尝试不同的代理">
    如果您没有获得想要的结果，请尝试切换到不同的代理。每个都有独特的方法。
  </Accordion>

  <Accordion title="对复杂请求要有耐心">
