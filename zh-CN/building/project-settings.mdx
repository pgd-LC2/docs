## 访问项目设置

点击编辑器中的设置图标以访问所有项目配置选项。

## 常规设置

### 项目信息

配置基本项目详细信息：

- **项目名称** - 识别您的项目
- **描述** - 添加有关项目的注释
- **标签** - 使用标签组织项目
- **可见性** - 设置为公开或私有

### 通知

控制通知首选项：

- **构建完成** - 构建完成时提醒
- **部署状态** - 发布成功/失败时通知
- **错误警报** - 获取关键错误通知
- **音效** - 启用/禁用音频通知

## 域名设置

### 主域名

设置您网站的主 URL：

1. 选择子域名（例如，`mysite.aisitebuilder.app`）
2. 或连接自定义域名
3. 设置为主域名

### 域名别名

添加重定向到主域名的其他域名：

- 添加多个自定义域名
- 配置重定向
- 管理 SSL 证书

<Tip>
  所有域名的 SSL 证书都会自动生成和续订。
</Tip>

## 托管设置

### 部署

配置网站的部署方式：

- **自动部署** - 自动发布更改
- **手动部署** - 需要手动批准
- **分支部署** - 从特定分支部署

### 性能

优化您的网站性能：

- **CDN** - 启用内容分发网络
- **缓存** - 配置缓存设置
- **压缩** - 启用 gzip/brotli 压缩
- **图像优化** - 自动图像压缩

## 分析

### 内置分析

跟踪您的网站访问者：

- **页面浏览量** - 查看哪些页面受欢迎
- **访问者来源** - 了解流量来自哪里
- **地理数据** - 查看访问者位置
- **设备类型** - 桌面与移动流量

### 外部分析

集成第三方分析：

- Google Analytics
- Plausible
- Fathom
- 自定义跟踪脚本

## 身份验证

### 用户身份验证

在您的网站上启用用户账户：

- **电子邮件/密码** - 传统身份验证
- **社交登录** - Google、GitHub 等
- **魔术链接** - 无密码电子邮件登录
- **双因素身份验证** - 增强安全性

### 访问控制

限制对您网站的访问：

- **密码保护** - 需要密码才能查看
- **IP 白名单** - 允许特定 IP 地址
- **仅限会员** - 需要用户账户

## 数据库

### 数据库配置

管理您网站的数据库：

- **查看表** - 查看所有数据库表
- **添加表** - 创建新表
- **编辑架构** - 修改表结构
- **导出数据** - 下载数据库内容

### 备份

保护您的数据：

- **自动备份** - 默认启用每日备份
- **手动备份** - 创建备份快照
- **恢复** - 恢复到以前的备份

## 文件存储

### 存储设置

配置文件上传和存储：

- **存储限制** - 查看可用存储
- **允许的文件类型** - 限制上传类型
- **最大文件大小** - 设置上传大小限制
- **CDN 交付** - 通过 CDN 提供文件

### 文件管理

组织上传的文件：

- 浏览所有上传的文件
- 创建文件夹
- 删除未使用的文件
- 查看存储使用情况

## 环境变量

### 密钥管理

安全存储敏感配置：

- **API 密钥** - 存储第三方 API 密钥
- **数据库凭据** - 连接字符串
- **OAuth 密钥** - 身份验证凭据
- **自定义变量** - 任何敏感数据

<Warning>
  切勿将密钥提交到代码中。始终使用环境变量。
</Warning>

## 版本历史

### 回滚和恢复

管理您的网站版本：

- **查看历史** - 查看所有以前的版本
- **比较版本** - 查看更改内容
- **恢复版本** - 恢复到以前的状态
- **预览版本** - 在不恢复的情况下查看

### 备份策略

创建战略备份：

- 在重大更改之前
- 完成功能后
- 在实验之前
- 定期间隔

## 高级设置

### 自定义代码

添加自定义功能：

- **自定义 CSS** - 覆盖样式
- **自定义 JavaScript** - 添加功能
- **头部标签** - 元标签、脚本
- **正文标签** - 分析、小部件

### SEO 配置

针对搜索引擎进行优化：

- **元标题** - 页面标题
- **元描述** - 页面描述
- **Open Graph** - 社交媒体预览
- **站点地图** - 自动生成的站点地图
- **Robots.txt** - 搜索引擎说明

### 性能监控

跟踪网站性能：

- **核心 Web 指标** - Google 性能指标
- **加载时间** - 页面速度跟踪
- **错误跟踪** - 监控错误
- **正常运行时间监控** - 跟踪可用性

## 团队设置

### 协作

与团队成员合作：

- **邀请成员** - 添加团队成员
- **角色和权限** - 控制访问
- **活动日志** - 查看谁更改了什么
- **评论** - 讨论更改

<Note>
  团队功能在 Pro 和 Enterprise 计划中可用。
</Note>
