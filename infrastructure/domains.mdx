## Domain options

### Default subdomain

Every project gets a free subdomain:

```
yoursite.aisitebuilder.app
```

This is automatically configured and ready to use immediately.

### Custom domains

Connect your own domain:

```
www.yourdomain.com
yourdomain.com
```

Custom domains give your website a professional appearance and build trust with visitors.

## Adding a custom domain

### Step 1: Purchase a domain

Buy a domain from a registrar:

- Namecheap
- GoDaddy
- Google Domains
- Cloudflare
- Any other registrar

### Step 2: Add domain to AISiteBuilder

1. Go to **Project Settings → Domains**
2. Click **Add Custom Domain**
3. Enter your domain name
4. Click **Add Domain**

### Step 3: Configure DNS

Add DNS records at your domain registrar:

<Tabs>
  <Tab title="Root domain">
    For `yourdomain.com`:

    | Type | Name | Value |
    |------|------|-------|
    | A | @ | `76.76.21.21` |
    | AAAA | @ | `2606:4700:3033::ac43:bd8d` |
  </Tab>

  <Tab title="Subdomain">
    For `www.yourdomain.com`:

    | Type | Name | Value |
    |------|------|-------|
    | CNAME | www | `cname.aisitebuilder.app` |
  </Tab>

  <Tab title="Both">
    For both `yourdomain.com` and `www.yourdomain.com`:

    | Type | Name | Value |
    |------|------|-------|
    | A | @ | `76.76.21.21` |
    | AAAA | @ | `2606:4700:3033::ac43:bd8d` |
    | CNAME | www | `cname.aisitebuilder.app` |
  </Tab>
</Tabs>

### Step 4: Verify domain

1. Wait for DNS propagation (5 minutes to 48 hours)
2. AISiteBuilder will automatically verify your domain
3. SSL certificate is generated automatically
4. Your website is live on your custom domain

<Tip>
  DNS changes typically propagate within 15-30 minutes, but can take up to 48 hours in some cases.
</Tip>

## SSL certificates

### Automatic SSL

SSL certificates are automatically:

- Generated when you add a domain
- Renewed before expiration
- Applied to all domains

Your website is always served over HTTPS.

### Manual SSL request

If automatic generation fails:

1. Go to **Project Settings → Domains**
2. Find your domain
3. Click **Request SSL Certificate**
4. Certificate will be generated

## Domain management

### Primary domain

Set which domain is your main URL:

1. Go to **Project Settings → Domains**
2. Click **Set as Primary** on your preferred domain
3. Other domains will redirect to primary

### Domain aliases

Add multiple domains that point to your website:

- All aliases redirect to primary domain
- Useful for multiple TLDs (.com, .net, .io)
- Maintain SEO with proper redirects

### Removing domains

Delete domains you no longer need:

1. Go to **Project Settings → Domains**
2. Click **Remove** next to the domain
3. Confirm removal

<Warning>
  Removing a domain will make your website inaccessible at that URL.
</Warning>

## Subdomains

### Using subdomains

Create subdomains for different purposes:

```
blog.yourdomain.com
shop.yourdomain.com
app.yourdomain.com
```

### Adding subdomains

1. Add subdomain in AISiteBuilder
2. Create CNAME record at your registrar:

| Type | Name | Value |
|------|------|-------|
| CNAME | blog | `cname.aisitebuilder.app` |

### Subdomain as primary

You can use a subdomain as your primary domain:

1. Add subdomain
2. Set as primary domain
3. Configure DNS

## DNS configuration guides

### Namecheap

1. Log in to Namecheap
2. Go to **Domain List**
3. Click **Manage** next to your domain
4. Go to **Advanced DNS** tab
5. Add the DNS records
6. Save changes

### GoDaddy

1. Log in to GoDaddy
2. Go to **My Products**
3. Click **DNS** next to your domain
4. Add the DNS records
5. Save changes

### Cloudflare

1. Log in to Cloudflare
2. Select your domain
3. Go to **DNS** tab
4. Add the DNS records
5. Ensure proxy status is **DNS only** (gray cloud)

### Google Domains

1. Log in to Google Domains
2. Select your domain
3. Go to **DNS** tab
4. Add the DNS records
5. Save changes

## Troubleshooting

<AccordionGroup>
  <Accordion title="Domain not connecting">
    If your domain isn't connecting:
    - Verify DNS records are correct
    - Wait for DNS propagation (up to 48 hours)
    - Check for typos in DNS configuration
    - Ensure no conflicting DNS records exist
  </Accordion>

  <Accordion title="SSL certificate not generating">
    If SSL isn't working:
    - Verify domain is properly connected
    - Wait 10-15 minutes after DNS propagation
    - Try manual SSL request
    - Check that DNS records point to correct IPs
  </Accordion>

  <Accordion title="Redirect loops">
    If you're experiencing redirect loops:
    - Check that only one domain is set as primary
    - Verify no conflicting redirects in DNS
    - Clear browser cache
    - Check for redirect rules at registrar
  </Accordion>

  <Accordion title="DNS propagation check">
    To check DNS propagation:
    - Use [whatsmydns.net](https://whatsmydns.net)
    - Enter your domain
    - Check if records are visible globally
  </Accordion>
</AccordionGroup>
