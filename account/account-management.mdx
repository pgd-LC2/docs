## 账户设置

### 个人资料信息

更新您的账户详细信息：

1. 转到 **账户设置 → 个人资料**
2. 编辑您的信息：
   - 显示名称
   - 电子邮件地址
   - 个人资料照片
   - 简介
3. 点击 **保存更改**

### 电子邮件偏好

管理电子邮件通知：

- **产品更新** - 新功能和改进
- **账单通知** - 付款和发票警报
- **营销电子邮件** - 提示、教程和促销
- **安全警报** - 登录和安全通知

<Tip>
  为了账户安全，无法禁用安全警报。
</Tip>

### 密码和安全

#### 更改密码

更新您的密码：

1. 转到 **账户设置 → 安全**
2. 点击 **更改密码**
3. 输入当前密码
4. 输入新密码
5. 确认新密码
6. 点击 **更新密码**

#### 双因素身份验证

启用 2FA 以增强安全性：

1. 转到 **账户设置 → 安全**
2. 点击 **启用 2FA**
3. 使用身份验证器应用扫描二维码
4. 输入验证码
5. 保存备份代码

支持的身份验证器应用：
- Google Authenticator
- Authy
- 1Password
- Microsoft Authenticator

#### 备份代码

生成用于账户恢复的备份代码：

1. 转到 **账户设置 → 安全**
2. 点击 **生成备份代码**
3. 下载并安全保存代码
4. 如果您失去对 2FA 设备的访问权限，请使用

<Warning>
  将备份代码存储在安全位置。每个代码只能使用一次。
</Warning>

## 登录方法

### 电子邮件和密码

标准登录方法：

- 使用您注册的电子邮件
- 输入您的密码
- 可选：启用"记住我"

### 社交登录

连接社交账户以快速登录：

- **Google** - 使用 Google 账户登录
- **GitHub** - 使用 GitHub 账户登录
- **Microsoft** - 使用 Microsoft 账户登录

#### 添加社交登录

1. 转到 **账户设置 → 已连接账户**
2. 点击所需服务旁边的 **连接**
3. 授权 AISiteBuilder
4. 账户已链接

#### 删除社交登录

1. 转到 **账户设置 → 已连接账户**
2. 点击服务旁边的 **断开连接**
3. 确认断开连接

<Note>
  在断开连接之前，请确保您至少启用了一种登录方法。
</Note>

## 会话管理

### 活动会话

查看和管理活动会话：

1. 转到 **账户设置 → 安全**
2. 查看 **活动会话**
3. 查看设备、位置和最后活动
4. 点击 **撤销** 以结束会话

### 会话超时

配置自动注销：

- **从不** - 无限期保持登录
- **1 小时** - 1 小时不活动后注销
- **1 天** - 1 天后注销
- **1 周** - 1 周后注销

## 数据和隐私

### 数据导出

下载您的账户数据：

1. 转到 **账户设置 → 隐私**
2. 点击 **导出数据**
3. 选择要导出的数据：
   - 项目和网站
   - 账户信息
   - 使用历史
4. 点击 **请求导出**
5. 通过电子邮件接收下载链接（24 小时内）

### 数据删除

请求删除账户数据：

1. 转到 **账户设置 → 隐私**
2. 点击 **删除数据**
3. 选择要删除的数据
4. 确认删除

<Warning>
  已删除的数据无法恢复。首先导出重要数据。
</Warning>

### 隐私设置

控制数据使用：

- **分析** - 允许使用分析
- **个性化** - 个性化推荐
- **第三方共享** - 与合作伙伴共享数据（默认关闭）

## 账户删除

### 删除您的账户

永久删除您的账户：

1. 转到 **账户设置 → 账户**
2. 点击 **删除账户**
3. 输入密码以确认
4. 提供反馈（可选）
5. 点击 **永久删除账户**

**会发生什么：**
- 所有网站都将取消发布
- 所有数据将在 30 天内删除
- 订阅被取消
- 剩余时间不退款

**宽限期：**
- 30 天宽限期以恢复账户
- 联系支持以在 30 天内恢复
- 30 天后，删除是永久性的

## API 访问

### API 密钥

生成用于程序化访问的 API 密钥：

1. 转到 **账户设置 → API**
2. 点击 **生成 API 密钥**
3. 命名您的密钥
4. 设置权限
5. 复制并安全保存密钥

<Warning>
  API 密钥提供完全的账户访问权限。切勿共享或提交到代码存储库。
</Warning>

### 管理 API 密钥

- **查看密钥** - 查看所有活动密钥
- **撤销密钥** - 禁用受损密钥
- **轮换密钥** - 定期生成新密钥
- **设置到期** - 在设定时间后自动使密钥过期

## 通知

### 应用内通知

配置应用内警报：

- **构建完成** - 构建完成时
- **部署状态** - 发布成功/失败
- **错误** - 关键错误
- **评论** - 团队成员评论
- **提及** - 当您被提及时

### 电子邮件通知

控制电子邮件频率：

- **实时** - 立即发送电子邮件
- **每日摘要** - 每天一次摘要
- **每周摘要** - 每周一次摘要
- **关闭** - 无电子邮件（安全除外）

### 推送通知

启用浏览器推送通知：

1. 转到 **账户设置 → 通知**
2. 点击 **启用推送通知**
3. 在浏览器提示中允许
4. 配置通知类型

### 编辑器偏好

自定义您的编辑器体验：

- **主题** - 浅色、深色或自动
- **字体大小** - 编辑器字体大小
- **行号** - 显示/隐藏行号
- **自动保存** - 启用自动保存
- **键盘快捷键** - 自定义快捷键

### 语言

设置界面语言：

- 英语
- 西班牙语
- 法语
- 德语
- 中文
- 日语
- 更多语言即将推出

### 时区

设置您的时区以获得准确的时间戳：

1. 转到 **账户设置 → 偏好**
2. 选择时区
3. 点击 **保存**

## 故障排除

<AccordionGroup>
  <Accordion title="无法登录">
    如果您无法访问您的账户：
    - 验证电子邮件地址是否正确
    - 使用"忘记密码"重置密码
    - 检查是否启用了 2FA
    - 如果问题仍然存在，请联系支持
  </Accordion>

  <Accordion title="未收到电子邮件">
    如果您没有收到电子邮件：
    - 检查垃圾邮件/垃圾文件夹
    - 验证设置中的电子邮件地址
    - 将 noreply@aisitebuilder.com 添加到联系人
    - 检查电子邮件偏好是否已启用
  </Accordion>

  <Accordion title="丢失 2FA 设备">
    如果您丢失了 2FA 设备：
    - 使用备份代码登录
    - 在安全设置中禁用 2FA
    - 在新设备上设置 2FA
    - 如果没有备份代码，请联系支持
  </Accordion>

  <Accordion title="账户被盗用">
    如果您的账户被盗用：
    - 立即更改密码
    - 撤销所有活动会话
    - 启用 2FA
    - 查看最近的活动
    - 联系支持
  </Accordion>
</AccordionGroup>
