## Account settings

### Profile information

Update your account details:

1. Go to **Account Settings → Profile**
2. Edit your information:
   - Display name
   - Email address
   - Profile photo
   - Bio
3. Click **Save Changes**

### Email preferences

Manage email notifications:

- **Product updates** - New features and improvements
- **Billing notifications** - Payment and invoice alerts
- **Marketing emails** - Tips, tutorials, and promotions
- **Security alerts** - Login and security notifications

<Tip>
  Security alerts cannot be disabled for account safety.
</Tip>

### Password and security

#### Changing password

Update your password:

1. Go to **Account Settings → Security**
2. Click **Change Password**
3. Enter current password
4. Enter new password
5. Confirm new password
6. Click **Update Password**

#### Two-factor authentication

Enable 2FA for enhanced security:

1. Go to **Account Settings → Security**
2. Click **Enable 2FA**
3. Scan QR code with authenticator app
4. Enter verification code
5. Save backup codes

Supported authenticator apps:
- Google Authenticator
- Authy
- 1Password
- Microsoft Authenticator

#### Backup codes

Generate backup codes for account recovery:

1. Go to **Account Settings → Security**
2. Click **Generate Backup Codes**
3. Download and save codes securely
4. Use if you lose access to 2FA device

<Warning>
  Store backup codes in a secure location. Each code can only be used once.
</Warning>

## Login methods

### Email and password

Standard login method:

- Use your registered email
- Enter your password
- Optional: Enable "Remember me"

### Social login

Connect social accounts for quick login:

- **Google** - Sign in with Google account
- **GitHub** - Sign in with GitHub account
- **Microsoft** - Sign in with Microsoft account

#### Adding social login

1. Go to **Account Settings → Connected Accounts**
2. Click **Connect** next to desired service
3. Authorize AISiteBuilder
4. Account is linked

#### Removing social login

1. Go to **Account Settings → Connected Accounts**
2. Click **Disconnect** next to service
3. Confirm disconnection

<Note>
  Ensure you have at least one login method enabled before disconnecting.
</Note>

## Session management

### Active sessions

View and manage active sessions:

1. Go to **Account Settings → Security**
2. View **Active Sessions**
3. See device, location, and last activity
4. Click **Revoke** to end session

### Session timeout

Configure automatic logout:

- **Never** - Stay logged in indefinitely
- **1 hour** - Logout after 1 hour of inactivity
- **1 day** - Logout after 1 day
- **1 week** - Logout after 1 week

## Data and privacy

### Data export

Download your account data:

1. Go to **Account Settings → Privacy**
2. Click **Export Data**
3. Select data to export:
   - Projects and websites
   - Account information
   - Usage history
4. Click **Request Export**
5. Receive download link via email (within 24 hours)

### Data deletion

Request account data deletion:

1. Go to **Account Settings → Privacy**
2. Click **Delete Data**
3. Select data to delete
4. Confirm deletion

<Warning>
  Deleted data cannot be recovered. Export important data first.
</Warning>

### Privacy settings

Control data usage:

- **Analytics** - Allow usage analytics
- **Personalization** - Personalized recommendations
- **Third-party sharing** - Share data with partners (off by default)

## Account deletion

### Deleting your account

Permanently delete your account:

1. Go to **Account Settings → Account**
2. Click **Delete Account**
3. Enter password to confirm
4. Provide feedback (optional)
5. Click **Permanently Delete Account**

**What happens:**
- All websites are unpublished
- All data is deleted within 30 days
- Subscriptions are canceled
- No refunds for remaining time

**Grace period:**
- 30-day grace period to recover account
- Contact support to restore within 30 days
- After 30 days, deletion is permanent

## API access

### API keys

Generate API keys for programmatic access:

1. Go to **Account Settings → API**
2. Click **Generate API Key**
3. Name your key
4. Set permissions
5. Copy and save key securely

<Warning>
  API keys provide full account access. Never share or commit to code repositories.
</Warning>

### Managing API keys

- **View keys** - See all active keys
- **Revoke keys** - Disable compromised keys
- **Rotate keys** - Generate new keys periodically
- **Set expiration** - Auto-expire keys after set time

## Troubleshooting

<AccordionGroup>
  <Accordion title="Can't log in">
    If you can't access your account:
    - Verify email address is correct
    - Reset password using "Forgot Password"
    - Check if 2FA is enabled
    - Contact support if issues persist
  </Accordion>

  <Accordion title="Not receiving emails">
    If you're not getting emails:
    - Check spam/junk folder
    - Verify email address in settings
    - Add noreply@aisitebuilder.com to contacts
    - Check email preferences are enabled
  </Accordion>

  <Accordion title="Lost 2FA device">
    If you lost your 2FA device:
    - Use backup codes to log in
    - Disable 2FA in security settings
    - Set up 2FA on new device
    - Contact support if no backup codes
  </Accordion>

  <Accordion title="Account compromised">
    If your account was compromised:
    - Change password immediately
    - Revoke all active sessions
    - Enable 2FA
    - Review recent activity
    - Contact support
  </Accordion>
</AccordionGroup>
