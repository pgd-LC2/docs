## 管理您的订阅

### 升级

随时升级您的计划：

1. 转到 **账户设置 → 账单**
2. 点击 **升级计划**
3. 选择您的新计划
4. 输入付款信息
5. 确认升级

您的新计划立即激活。

### 降级

在账单周期结束时降级：

1. 转到 **账户设置 → 账单**
2. 点击 **更改计划**
3. 选择较低层级
4. 确认降级

更改在当前账单周期结束时生效。

<Warning>
  降级可能会限制对功能的访问。确保您的使用符合新计划限制。
</Warning>

### 取消

随时取消您的订阅：

1. 转到 **账户设置 → 账单**
2. 点击 **取消订阅**
3. 提供反馈（可选）
4. 确认取消

您将保留访问权限直到账单周期结束。

## 账单

### 付款方式

接受的付款方式：

- **信用卡** - Visa、Mastercard、American Express
- **借记卡**
- **PayPal**
- **银行转账**（仅限 Enterprise）

### 更新付款方式

更改您的付款信息：

1. 转到 **账户设置 → 账单**
2. 点击 **付款方式**
3. 添加新付款方式
4. 设置为默认
5. 删除旧方式（可选）

### 账单周期

选择您的账单频率：

- **月度** - 每月账单
- **年度** - 每年账单一次（节省 17%）

### 发票

访问您的账单历史：

1. 转到 **账户设置 → 账单**
2. 点击 **发票**
3. 查看或下载发票
4. 将发票通过电子邮件发送给会计

4. 确认删除

## 账单问题

### 付款失败

如果付款失败：

1. 您将收到电子邮件通知
2. 更新付款方式
3. 手动重试付款
4. 或等待自动重试（3 次尝试）

3 次失败尝试后，您的账户可能会被暂停。

### 退款

退款政策：

- 新订阅 **30 天退款保证**
- 年度计划 **按比例退款**（前 30 天）
- 令牌购买 **不退款**
- 30 天后 **不退款**

请求退款：

1. 联系支持
2. 提供原因
3. 退款在 5-7 个工作日内处理

### 账单争议

对收费提出异议：

1. 使用发票号码联系支持
2. 解释问题
3. 提供支持信息
4. 3 个工作日内解决

## 税务信息

### 销售税

根据您的位置可能适用销售税：

- **美国客户** - 州销售税
- **欧盟客户** - 增值税
- **其他地区** - 适用的当地税

### 免税

如果您免税：

1. 转到 **账户设置 → 账单**
2. 点击 **税务信息**
3. 上传免税证书
4. 提交审查

2 个工作日内批准。

### 税务发票

下载税务发票：

1. 转到 **账户设置 → 账单**
2. 点击 **发票**
3. 选择发票
4. 下载带有税务详细信息的 PDF
